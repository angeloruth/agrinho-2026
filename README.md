**Tecnologias:** HTML5, CSS3, JavaScript puro (sem frameworks)

---

## 🎯 Objetivo

Este projeto é um site educativo sobre sustentabilidade agrícola. O objetivo é:

1. Ensinar sobre práticas agrícolas sustentáveis
2. Mostrar tecnologias inovadoras no agronegócio
3. Inspirar com exemplos reais de sucesso
4. Engajar com conteúdo interativo (abas, quiz, formulário)
5. Promover consciência ambiental

## 🏗️ Como o Projeto Funciona

### 1. HTML (index.html)

O site tem **9 seções principais**:

1. **Navegação** - Menu no topo com links para as seções
2. **Hero** - Imagem grande com título principal
3. **Sobre** - 3 abas: Importância, Desafios, Soluções
4. **Tecnologia** - 6 cards com tecnologias agrícolas
5. **Sustentabilidade** - 5 práticas com seleção interativa
6. **Casos de Sucesso** - 3 exemplos reais de fazendas
7. **Educação** - Quiz com 3 questões
8. **Contato** - Formulário para enviar mensagens
9. **Footer** - Rodapé com informações

### 2. CSS (styles.css)

O CSS organiza a aparência do site:

- **Cores:** Verde para sustentabilidade (#4CAF50)
- **Fontes:** Poppins (títulos) e Raleway (corpo)
- **Layout:** Responsivo para desktop, tablet e mobile
- **Animações:** Transições suaves ao clicar e rolar
- **Sem inline CSS:** Todo o CSS está no arquivo externo

### 3. JavaScript (script.js)

O JavaScript torna o site interativo:

- **Menu mobile:** Abre e fecha ao clicar
- **Navbar:** Muda de cor ao rolar
- **Abas:** Mostra conteúdo diferente ao clicar
- **Práticas:** Destaca a prática selecionada
- **Quiz:** Calcula pontuação e mostra resultado
- **Formulário:** Valida e envia mensagem
- **Botão voltar:** Aparece ao rolar para baixo

## ✨ Funcionalidades

### Menu Responsivo
- Desktop: Menu visível no topo
- Mobile: Menu hambúrguer (☰)

### Abas Interativas
- Clique em cada aba para ver conteúdo diferente
- Animação suave ao trocar

### Seleção de Práticas
- Clique em cada prática para destacá-la
- Mostra o impacto ambiental

### Quiz Gamificado
- 3 perguntas sobre sustentabilidade
- Múltipla escolha
- Mostra pontuação ao final

### Formulário de Contato
- Campos: Nome, Email, Mensagem
- Valida se todos os campos estão preenchidos
- Mostra mensagem de sucesso

### Botão Voltar ao Topo
- Aparece após rolar 300px
- Clique para voltar suavemente ao topo
